# Инструкция по установке GoRanked Chat Desk для Windows

## Что нужно скачать

Для Windows нужно скачать файл установщика (`.exe` файл) с GitHub Releases.

### Шаги установки:

1. **Откройте страницу релизов:**
   - Перейдите по ссылке: https://github.com/Revoool/chat.goranked/releases/latest
   - Или откройте: https://github.com/Revoool/chat.goranked/releases

2. **Найдите файл для Windows:**
   - Ищите файл с названием: `GoRanked Chat Desk-Setup-1.0.19.exe`
   - Или файл с расширением `.exe` в разделе "Assets"

3. **Скачайте файл:**
   - Нажмите на файл `.exe` для скачивания
   - Дождитесь завершения загрузки

4. **Установите приложение:**
   - Откройте скачанный файл `.exe`
   - Если Windows показывает предупреждение безопасности:
     - Нажмите "Подробнее" (More info)
     - Затем "Выполнить в любом случае" (Run anyway)
   - Следуйте инструкциям установщика
   - Приложение будет установлено в папку по умолчанию (обычно `C:\Users\ВашеИмя\AppData\Local\Programs\goranked-chat-desk`)

5. **Запустите приложение:**
   - После установки приложение можно запустить из меню "Пуск"
   - Или найдите ярлык на рабочем столе

## Автоматические обновления

После установки приложение будет автоматически проверять наличие обновлений:
- При запуске приложения
- По кнопке "Перевірити оновлення" в настройках
- Обновления будут загружаться и устанавливаться автоматически

## Системные требования

- Windows 10 или новее
- Процессор: x64 (64-bit)
- Минимум 200 МБ свободного места на диске

## Проблемы при установке?

Если Windows блокирует установку:
1. Щелкните правой кнопкой мыши на файл `.exe`
2. Выберите "Свойства"
3. Внизу окна нажмите "Разблокировать" (Unblock)
4. Нажмите "ОК"
5. Запустите установщик снова
